What are the differences between var, let, and const?

Var: Var is mutable means values of var can be changed. Var is globally scoped.
Let: Let is almost similar to var means it is also mutable and the value of let can be changed. But let is block scoped.
Const: Const is immutable means its value cannot be changed or re-assigned. It is also block scoped as let.

What is template literals in javascript?

Template literal is a new feature in javascript. It is introduced in ES6. It is also known as string literal. It allows us to perform the embedded expression in a string whereas it was not possible in a regular string. We can define a template string by using backtick(` `) instead of using single or double quotes. Now we can use any expression in this by using ${} and it is absolutely fine.

What are the differences between map, foreach, filter and find

.map() loop through every element in an array and returns a new array with the updated elements.
.forEach() is used to loop through on every element in an array but it does not change the original array and it always returns undefined if we try to return anything.
.filter() loop through every element in an array to check if it fulfils the criteria and returns a new array with the elements that meets the condition.
.find() loop through the element in an array and it returns the first element which meets the condition.
